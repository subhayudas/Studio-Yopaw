What Was Built — High-Level Summary

New Feature

Mat Rental Step

Yin yoga bookings now ask whether the customer has their own mat before letting them pick a date. Choosing "rent" adds $5 + taxes to the order total.

Rewrite

Availability & Breeds API

Both APIs were rewritten from reading Square Catalog / all-day bookings to reading a flat class-schedule.json file — far simpler and no extra Square API calls.

Integration

Zapier Lead Capture

Three Zapier webhooks fire on key events (new Square customer, confirmed booking, contact form submit) so leads flow into Zapier automations even before payment.

Integration

Google Ads + Clarity

Google Ads conversion tracking fires when the customer submits the contact form. Microsoft Clarity session recording is active on all pages.

Bug Fix

Payment Resilience

If payment fails after the booking is created, the booking is immediately cancelled to prevent ghost appointments. User sees a human-readable error instead of a raw Square code.

Removed

UI Cleanup

The Testimonials section was removed from the landing page. The "Reviews" link was removed from Navbar and Footer. The pricing card header was simplified during booking steps.
